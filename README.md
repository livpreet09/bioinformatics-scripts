# Bioinformatics Sequence Analysis Toolkit
A collection of Python functions for DNA sequence analysis, built as part of a self-directed bioinformatics learning project.

## Functions Included
- **count_nucleotides** — counts A, T, G, C in a DNA sequence
- **transcribe** — converts DNA to RNA (T → U)
- **reverse_complement** — returns the reverse complement of a DNA strand
- **gc_content** — calculates GC content percentage
- **find_motif** — finds all positions of a motif in a DNA sequence
- **parse_fasta** — parses a FASTA file into a Python dictionary

## Skills Demonstrated
Python · Biopython · String manipulation · File parsing · Sequence analysis · FASTA format
